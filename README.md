# 图片字幕生成器

一个面向内容创作者的浏览器端字幕图片工具。它可以把单张图片快速处理成带电影感字幕的分享图，也可以把多张已有字幕截图拼接成适合收藏和发布的台词长图。

项目采用纯前端实现，打开 `index.html` 就能使用，不依赖后端服务，也不会把图片上传到服务器。

## 在线预览

访问：[https://hovz-subgen.netlify.app/](https://hovz-subgen.netlify.app/)

## 作品亮点

- **两种创作模式**：支持单图字幕生成和多图台词拼接。
- **实时预览**：上传图片、调整字幕参数后即时生成画布预览。
- **字幕样式可控**：可调整字号、行高、字体、颜色、描边、背景强度、对齐方式和分隔线。
- **双语字幕**：中文与英文可分别输入，并支持英文比例和字体设置。
- **比例适配**：支持原始比例、3:4、1:1、9:16、16:9、4:3 等常见发布尺寸。
- **水印与导出**：可添加自定义水印，并导出为 PNG 图片。
- **本地处理**：所有图片处理都在浏览器 Canvas 中完成，适合轻量、隐私友好的使用场景。

## 快速使用

直接双击或用浏览器打开：

```text
index.html
```

也可以在项目目录启动一个本地静态服务：

```bash
python -m http.server 4173
```

然后访问：

```text
http://localhost:4173
```

## 使用流程

1. 选择创作方式：`字幕生成` 或 `台词拼接`。
2. 上传一张或多张图片。
3. 输入自定义字幕、双语字幕，或使用内置灵感文案。
4. 调整字幕样式、图片比例和水印。
5. 在右侧预览区确认效果。
6. 点击 `保存图片` 导出 PNG。

## 技术实现

- HTML / CSS / JavaScript 单文件应用
- Canvas 负责图片绘制、字幕叠加、拼接和导出
- FileReader 与 Image 对象负责本地图片读取
- 无构建流程、无运行时依赖、无后端接口
