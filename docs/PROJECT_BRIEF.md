# 项目说明：图片字幕生成器

## 项目定位

图片字幕生成器是一个轻量的内容创作工具，面向需要快速制作社交媒体配图、电影感字幕图、台词长图和双语字幕图的用户。它强调“打开即用”和“本地处理”，让用户不用安装软件，也不用把素材上传到远程服务。

## 核心用户场景

- 给旅行、日常、美食、情绪类照片添加一句或多句字幕。
- 为短视频封面、朋友圈、小红书、微博等场景快速生成带字幕图片。
- 将多张已有台词截图拼接成一张长图，便于收藏或分享。
- 制作中英双语字幕图，用于语录、影视台词或语言学习内容。

## 功能设计

### 字幕生成

用户上传单张图片后，可以输入自定义字幕、使用内置灵感文案，或分别输入中英双语内容。工具会在 Canvas 中绘制图片、渐变遮罩、字幕背景、描边文字和可选水印，并实时呈现在预览区域。

### 台词拼接

用户可以上传多张字幕截图。首图完整保留，后续图片按设置截取底部台词条并拼接成一张长图，适合制作影视台词合集。

### 样式控制

控制面板提供字幕行高、字号、背景强度、描边宽度、字体、对齐、分隔线、水印和输出比例等设置。目标是让非设计背景的用户也能快速得到可发布的视觉效果。

## 技术实现

- 使用原生 HTML、CSS 和 JavaScript 构建，无框架依赖。
- 使用 `FileReader` 读取本地图片文件。
- 使用 `Image` 对象加载素材并获取原始尺寸。
- 使用 Canvas API 完成图片裁切、等比填充、字幕绘制、拼接和 PNG 导出。
- 使用浏览器端状态变量管理当前模式、已上传图片、比例设置和实时预览状态。

## 作品价值

这个项目的价值不在复杂技术栈，而在完整的产品闭环：用户从上传素材、编辑字幕、调整视觉样式到导出成品，都可以在一个页面内完成。作为作品项目，它展示了前端工程中很实用的一类能力：把浏览器能力组织成一个可用、可交互、可交付的小工具。

## 可继续扩展

- 增加预设模板，如电影字幕、日系留白、社交封面等。
- 支持批量导出多张字幕图。
- 增加截图示例和在线演示地址。
- 将 CSS、渲染逻辑和状态管理拆分成多个文件，方便长期维护。
- 增加更多文本自动换行和超长字幕适配策略。
